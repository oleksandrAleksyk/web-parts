Navbar Structure

Navbar consist of three part. v.0.1 use the following structure Company Name - Navbar Items - Additional Bar
To override each components use:
.navbar .company-name { display: inline; }
.navbar .navbar-items { font-color: red; }
.navbar .additional-bar { margin: 25px; }

Colors

You can choose between dark, light, blue, red, green, yellow and tranparent color schems. Example:
class="navbar dark"//class="navbar light"

Fonts

I use 3 font famillies: Roboto, Lato and Poppins. To use it add class name font. Example:
class="navbar font-roboto" //class="navbar font-lato"
